Getting started

Start using Angular elements or any other web components in your applications by following these 5 steps...

  1. Install npm i ngx-loadable
  2. Add import { LoadableModule } from 'ngx-loadable';
  3. Append LoadableMoudule to the imports: [] of your AppModule
  4. Create a component using ng generate ngx-loadable:module lazy
  5. Use ngx-loadable component to load the lazy component
